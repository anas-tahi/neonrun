# NEON RUN 🎮

**LIVE demo URL: https://neonrun.onrender.com **

**The 20-Stage Platform Challenge** — a browser platformer with full physics, 4 unique characters, and a brutal difficulty curve.

## Play it live
> Deployed on Render — link appears after deployment.

## Controls
| Key | Action |
|-----|--------|
| `A` / `←` | Move left |
| `D` / `→` | Move right |
| `Space` | Jump (press again = double jump) |
| `Shift` | Dash (invincible during dash!) |
| `ESC` | Quit to outro |

## Stages
- **1–10** Easy — learn the mechanics
- **11–16** Hard — moving platforms, spikes, wind, gravity shifts, time limits
- **17–20** Impossible — < 10% of players finish Stage 20

## Characters
- 🥷 **Ninja** — Fast, motion-blur dash, red scarf
- 🤖 **Robot** — Precise, glowing visor, blinking antenna
- 👻 **Ghost** — Floaty, glowing eyes, wavy body
- ⚔️ **Knight** — Armoured, plumed helmet, cape + shield

## Tech
Pure HTML + Canvas + JavaScript. No frameworks, no libraries. One file.
